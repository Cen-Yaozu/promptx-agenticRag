## 🎯 核心概念

DeeChat 基于"**一个空间一个对话**"的设计理念：

- **对话空间** - 不同的对话环境
- **知识库** - 文档和知识的存储系统
- **智能搜索** - 基于语义理解的搜索
- **AI助手** - 智能对话模型
