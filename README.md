## 🌟 核心特性

- **🎯 一个空间一个对话** - 简化的用户交互设计
- **📚 智能知识库** - 文档自动索引和语义搜索
- **🤖 多AI模型支持** - 支持多种AI提供商
- **🔒 隐私保护** - 本地部署，数据安全
- **🌍 多语言支持** - 完整的国际化支持
- **🚀 易于部署** - 支持Docker、Kubernetes等多种部署方式

## 🚀 快速开始

### 使用 Docker (推荐)

```bash
git clone https://github.com/deepractice/dee-chat.git
cd dee-chat
docker-compose up -d
```

### 本地开发

```bash
git clone https://github.com/deepractice/dee-chat.git
cd dee-chat
npm install
npm run dev
```

访问 http://localhost:3000 开始使用！

## 🎯 设计理念

DeeChat 基于"**一个空间一个对话**"的设计理念：

- **对话空间** - 不同的对话环境，每个空间专注一个主题
- **知识库** - 文档和知识的智能存储系统
- **智能搜索** - 基于语义理解的精准搜索
- **AI助手** - 友好的智能对话体验
